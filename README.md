# 🚢 Titanic Survival Prediction using Machine Learning
Exploratory Data Analysis and Machine Learning model to predict Titanic passenger survival using Python

---

## 📌 **Project Overview**

This project builds a **Machine Learning model** to predict passenger survival in the Titanic disaster. It follows a complete end-to-end **Data Science workflow** including data cleaning, exploratory data analysis (EDA), feature engineering, and classification modeling.

The objective is to identify key factors influencing survival and develop a predictive model using passenger attributes.

---

## 🎯 **Problem Statement**

Given passenger information such as age, gender, ticket class, fare, and family details, predict whether a passenger survived the Titanic disaster or not.

---

## 🛠️ **Technologies Used**

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Scikit-learn**

---

## 📊 **Project Workflow**

### **1. Data Cleaning**
- Handled missing values (Age, Embarked, etc.)
- Removed irrelevant or redundant features
- Converted categorical variables into numerical format

### **2. Exploratory Data Analysis (EDA)**
- Survival rate by gender
- Survival rate by passenger class
- Age distribution analysis
- Relationship between fare, class, and survival

### **3. Feature Engineering**
- Encoding categorical variables
- Selecting important features for prediction
- Preparing dataset for modeling

### **4. Model Building**
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

### **5. Model Evaluation**
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## 🤖 **Machine Learning Model**

- Train-Test Split applied  
- Multiple classification models tested  
- Best performing model selected based on evaluation metrics  

🏆 **Best Model:** Random Forest Classifier 

---

## 📊 **Key Insights**

- **Female passengers had significantly higher survival probability than males**
- **First-class passengers had the highest survival rate**
- **Passenger class (Pclass) was a strong predictor of survival**
- **Fare and age influenced survival likelihood**

---

## 📈 **Results**

- Model successfully identifies patterns influencing survival  
- Random Forest performed better compared to simpler models  
- Demonstrates effective application of classification techniques
