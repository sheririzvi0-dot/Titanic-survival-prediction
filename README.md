# Titanic-Survival-Prediction(ML MOdel)
Exploratory Data Analysis and Machine Learning model to predict Titanic passenger survival using Python

📌 Project Overview

This project applies Exploratory Data Analysis (EDA) and Machine Learning to predict whether a passenger survived the Titanic disaster or not.

The goal is to analyze passenger data, identify patterns, and build a predictive model using Python.


📊 Steps Performed

- Data cleaning and handling missing values
- Exploratory Data Analysis (EDA)
- Feature selection and engineering
- Model building using Machine Learning
- Model evaluation and prediction


🛠 Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

🤖 Machine Learning Model

- Logistic Regression (or Random Forest if you used it)
- Train-test split
- Accuracy evaluation


🎯 Objective

Predict whether a passenger survived (1) or not (0) based on features like age, gender, class, etc.
