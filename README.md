# 🚢 Titanic Survival Prediction using Machine Learning
Exploratory Data Analysis and Machine Learning model to predict Titanic passenger survival using Python



## **📌 Project Overview**

This project applies **Exploratory Data Analysis (EDA)** and **Machine Learning** to predict whether a passenger survived the Titanic disaster. Using historical passenger data, the project explores survival patterns, performs data preprocessing, and builds a predictive model capable of classifying passenger survival.

The project demonstrates the complete **Machine Learning pipeline**, from data preparation to model evaluation, while showcasing practical data science techniques.

---

## **🎯 Project Objectives**

- **Analyze passenger data to identify survival patterns**
- **Perform data cleaning and preprocessing**
- **Engineer and select relevant features**
- **Build a Machine Learning classification model**
- **Evaluate model performance using standard metrics**

## **📈 Key Insights**

- **Female passengers had a significantly higher survival rate than males.**
- **First-class passengers were more likely to survive than those in lower classes.**
- **Age and fare played an important role in survival probability.**
- **Passenger class and gender were among the strongest predictive features.**
